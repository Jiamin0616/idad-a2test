Bottle Orchestra — Progress & Analysis

Open Prototype ↗

Concept & Design Context

Prompt: Skeuomorphism with a sketchy/lab vibe. Hand-drawn bottles act as instruments: the fill level controls pitch (drag up/down), and a 16-step ring sets rhythm.

Style: transparent glass outlines + colored granular “liquid” (water, rice, beads, sand). Fixed hand-drawn bar sketch as background.

References: Room1 · Limp Body Beat

Technical Approach

  • Audio: Tone.js (AMSynth, PluckSynth, NoiseSynth for sand/rice).
  • Sequencing: shared 16-step Transport; each bottle has its own step pattern.
  • Pitch Map: fill% → scale degrees (C pentatonic / A natural minor / whole tone).
  • Advanced technique: Math.random() for seeding positions/materials/shapes.
  • Graphics: SVG path clip for liquid; glass is outline only; fixed background image.

System Map

User
 ├─ Drag bottle neck → position (x,y)
 ├─ Drag bottle body → fill% → pitch
 ├─ Click tiny squares → per-bottle steps
 └─ Play/Stop → Tone.Transport (shared clock)
        ├─ step tick (0..15)
        └─ for each bottle: if step active → trigger synth

Week 8 Peer Testing — Findings

Q1 · Playability / Triggering

  • Not obvious how to trigger (4): people didn’t realise to click the tiny squares.
  • Play button missed (3): discoverability issue; needs to stand out.
  • Sequencer dots small (2): understood, but hard to hit at first.
  • Multi-bottle model unclear (1): unsure about “playing together”.
  • Needs instructions (≈3): onboarding/help requested.
  • Positive: once discovered, it feels intuitive and fun (4).

Q2 · Visual Style & Feedback Clarity

  • Clearer instructions/indicators (5+): onboarding note and stronger cues.
  • Show bottle & loop together; clearer selection (1): selection feedback could differ.
  • Too much at once (1): simplify or show info contextually.
  • Label bottles / sound mapping (1): explicit labels help expectations.
  • Style theming (1): more unified/stylised theme = more engaging.
  • Positive: visual style is engaging (1).

User Test Tasks (the two prompts)

  2. Layer a texture: Add a second bottle with sand → set a faster pattern (8th-feel) → adjust fill so the two bottles harmonise.
